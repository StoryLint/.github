# StoryLint

Convert ambiguous product stories into deterministic, testable specifications.
Messy narrative in. Executable behavior out.
Make product requirements executable.

---

### What it is

- Chrome extension for transforming product requirements in-place
- AI-powered API layer normalization and behavioral extraction
- Structured output engine for deterministic acceptance criteria
- Admin portal for usage control and configuration

---

### What it does

- Extracts actors, actions, conditions, and outcomes
- Enforces one behavior per scenario
- Eliminates subjective language
- Generates clean **Given / When / Then** specifications
- Produces output suitable for Jira, Linear, or plain text
